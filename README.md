Student Information Management System using XML

Project Explanation 

This project is a Student Information System using XML.

It stores details of students in a structured and organized format.

This root element is <Student_Info>, which contains multiple <Student_Details> elements.

Each student has a unique ID and includes information like:

2. Gender                                   7. Address
